# Task 5: Interactive Business Dashboard in Streamlit

## Objective
Develop a dynamic, cloud-ready Business Intelligence (BI) dashboard using **Streamlit** to analyze financial metrics, category sales, and high-value customer distributions using the Global Superstore dataset.

## Features Implemented
- **Dynamic Filtering System:** Multi-select filtering options in the sidebar for `Region`, `Category`, and `Sub-Category` attributes.
- **Top-line Strategic KPIs:** Real-time summary metric boxes displaying Total Sales, Gross Profit, and Net Profit Margins.
- **Data Storytelling Graphs:** - Dual group bar charts comparing overall Sales vs Revenue across individual categories.
  - Horizontal rank plots charting the **Top 5 Customers by Sales Volume**.
- **Interactive Data Explorer:** Embedded dynamic records sheet providing clean access to raw operational data tables.

## Technologies Used
- **Streamlit** (UI Component Rendering)
- **Pandas** (Operational Data Wrangling)
- **Plotly Express** (Dynamic Visualizations)
